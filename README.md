开发环境 macOS 10.12.3 Chrome 56 nodejs 6.11.1

# 技术栈
vue2+axios+webpack

# 克隆到本地
```
https://github.com/Jacknode/Vue-delivery-demo.git
```

# 安装依赖

# 开启本地服务器localhost:8080
npm run dev

# 发布环境
npm run build
